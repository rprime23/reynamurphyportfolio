E-commerce UI Features & Interactive Modules

Work Dates: May 2023 - March 2026

A collection of production-ready UI features built with HTML and CSS for live e-commerce environments. These features were designed to support marketing campaigns, product pages, and add-to-cart experiences while remaining flexible within CMS-driven page structures.

These UI features were developed using HTML and CSS in NetSuite SuiteCommerce Advanced using the built-in CMS_HTML block. We were prohibited from using JavaScript for most of the features.

These UI features were developed under real production constraints, requiring careful attention to maintainable markup, reusable styling patterns, and performance considerations when deployed across multiple pages and campaigns.
