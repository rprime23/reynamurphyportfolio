Frog App

Project Dates: April 2023 - May 2023

Frog App is an educational app designed to teach about frogs and frog conservation. With interactive questions on tests and large reading sections, this app focuses on testing the users knowledge of frogs.

Frog App was created in a five person team. This project was built in QT Creator using C++ code. I contributed with code, UI design, and art.
